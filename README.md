# chat-input

## 简介

实现文本和图片在光标处输入、插入、粘贴、换行

## npm script

- `dev`: 打包并监听文件更改
- `build`: 打包
- `build:type`: 打包并在主目录生成js文件
- `clean`: 清理打包生成文件目录
- `link:add`: 创建软连接便于本地测试
- `link:remove`: 删除软连接


## 软连接使用方法

```js
npm run link:add
```

## 使用软连接

```js
npm link name
```

## 使用软连接

```js
npm run link:remove
```
